# ROM Checksums

This document contains verified checksums for known good dumps of ROMs documented in this wiki.
Checksums were generated from ROMs in the GoodSNES collection.

## Checksum Sources

- **GoodTools**: Cowering's verification tools (GoodSNES, GoodNES, etc.)
- **TOSEC**: The Old School Emulation Center
- **No-Intro**: DAT-o-MATIC verified dumps

## Notes

- `[!]` indicates a verified/good dump per GoodTools naming conventions
- All ROMs are headered (SNES standard format)
- Size is in bytes
- Hash values are lowercase hexadecimal
- Version numbers (V1.0, V1.1) indicate ROM revision
