


Bits and Bytes
	Bits in order, little endian:
		ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz
		ABCDEFGH_IJKLMNOP_QRSTUVWX_YZabcdef_ghijklmn_opqrstuv_wxyz
	Single Byte => big endian:
		HGFEDCBA
	Multiple bytes => little endian, single byte => big endian:
		HGFEDCBA PONMLKJI XWVUTSRQ fedcbaZY nmlkjihg vutsrqpo 0000zyxw

Bit -- One bit
Nibble -- Four bits
Byte -- Eight bits
Word -- Sixteen bits, 2 bytes
Long -- 24 bits, 3 bytes
Int -- 32 bits, 4 bytes
