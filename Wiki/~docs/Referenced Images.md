## File Types

| Extension | Description |
|-----------|-------------|
| `.png` | Screenshot or image file |
| `.xcf` | GIMP native format (editable source) |
| `.json` | Metadata file (coordinates, annotations) |
| `.jpg` | JPEG image (typically box art) |

## Notes

- Most `.png` files have corresponding `.xcf` (GIMP source) and `.json` (metadata) files
- Text Table images show the character encoding mapped in RAM for text decoding
- `[!]` in filenames indicates a verified/good dump per GoodTools naming conventions
