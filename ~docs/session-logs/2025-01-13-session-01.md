### Tools Created

1. **rom_labeler.py** (~500 lines)
   - ROM label/symbol management tool
   - Auto-detect subroutines from JSR/JSL instructions
   - Built-in Soul Blazer known labels (code, data, RAM, hardware)
   - Multi-format export:
     - Mesen (.mlb)
     - bsnes/higan (.sym)
     - CA65 assembler (.inc)
     - WLA-DX assembler (.sym)
     - JSON full database
   - LoROM address conversion utilities
   - Label type classification

2. **palette_viewer.py** (~700 lines)
   - SNES palette viewer and editor with GUI
   - 15-bit BGR color format support
   - Known Soul Blazer palette locations
   - Features:
     - Interactive color swatches
     - RGB sliders with SNES precision
     - Color picker integration
     - Palette effects (grayscale, invert, brighten, darken, rotate)
     - Export/import JASC .pal format
     - PNG image export
     - ROM palette scanning
     - CLI mode for batch operations

3. **sprite_sheet_exporter.py** (~500 lines)
   - Export sprites from ROM as organized sheets
   - Category filtering (player, enemy, npc, boss, item, effect)
   - Animation frame support
   - Configurable scale and layout
   - JSON metadata export
   - Known Soul Blazer sprite definitions
   - 4BPP tile decoding with palette application
