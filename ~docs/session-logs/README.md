# Session Logs

AI-assisted development session documentation.

## Purpose

Session logs capture:
- Work completed during each development session
- Decisions made and their rationale
- Technical findings and discoveries
- Next steps and remaining work

## Naming Convention

Files are named: `YYYY-MM-DD-session-NN.md` or `YYYY-MM-DD-session-NN-description.md`

Examples:
- `2025-12-30-session-01.md`
- `2025-12-31-session-02-dq3r-animations.md`
