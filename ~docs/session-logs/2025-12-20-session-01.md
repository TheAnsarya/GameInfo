### CSpell Configuration Fix

Added `ignoreRegExpList` patterns to `.cspell.json` to prevent false positive spell check warnings for hexadecimal values:

```json
"ignoreRegExpList": [
    "/\\$[0-9a-fA-F]+/g",    // $7ebfff (assembly style)
    "/0x[0-9a-fA-F]+/g",     // 0xABCD (C style)
    "/\\b[0-9a-fA-F]{2,}h\\b/g", // 1234h (Intel style)
    "/#[0-9a-fA-F]{3,8}\\b/g"    // #FFFFFF (color codes)
]
```
