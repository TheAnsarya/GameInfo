## Technical Decisions

1. **Framework**: .NET 10 (available SDK) with Blazor Interactive WebAssembly + Server prerendering
2. **Database**: SQLite with FTS5 for embedded full-text search
3. **Styling**: Custom CSS design system (no heavy frameworks)
4. **Content**: Static generation at build time from GameInfo repo
5. **Wiki**: External links to games.darkrepos.com (MediaWiki)

## Design System Summary

- **Colors**: Dark theme with earth tones (browns, greens) and cream accents
- **Typography**: Inter (body), JetBrains Mono (code)
- **Accessibility**: WCAG 2.1 AA minimum, AAA for text contrast
