### 1. TAS Converter - Full Read/Write Support (Commit: abca511)

Added write support to all previously read-only TAS formats:
- **LSMV** - ZIP archive creation for lsnes
- **MMO** - Added read support for Mesen2 text format
- **FCM/FMV** - Binary writers for legacy NES formats
- **PXM/PJM** - PlayStation format writers
- **YMV/MC2/DSM** - Saturn, PCE, NDS format writers
- **LMP** - Doom engine format writer

All 17 formats now support both reading and writing.
