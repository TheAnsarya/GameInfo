### DarkRepos CLI Tool

Created `DarkRepos.Build` CLI application:
- **import** - Import games from GameInfo repository
- **seed** - Seed database with sample data
- **info** - Show database information

Features:
- System.CommandLine for argument parsing
- Spectre.Console for rich terminal output
- ASCII art banner with FigletText
- Database statistics table
- Clear/force options for re-import

Usage examples:
```bash
darkrepos import -r C:\repos\GameInfo -d wiki.db -c
darkrepos seed -d wiki.db
darkrepos info -d wiki.db
```
