# Chat Logs

AI conversation history for context preservation.

## Purpose

Chat logs preserve:
- Full conversation history
- Questions asked and answers received
- Context for future sessions
- Reference for decisions made

## Naming Convention

Files are named: `YYYY-MM-DD-chat-NN.md`

Examples:
- `2025-12-08-chat-01.md`
- `2025-12-09-chat-01.md`

## Usage

Chat logs help:
1. Resume work after breaks
2. Reference past discussions
3. Understand decision context
4. Track project evolution
