## Technical Notes

### CSpell Regex Patterns

The regex patterns in `.cspell.json`:
```json
"ignoreRegExpList": [
    "/\\$[0-9a-fA-F]+/g",
    "/0x[0-9a-fA-F]+/g",
    "/\\b[0-9a-fA-F]{2,}h\\b/g",
    "/#[0-9a-fA-F]{3,8}\\b/g"
]
```

These match:
- `$7ebfff`, `$00`, `$FFFF` - Assembly/ROM addresses
- `0xABCD`, `0x1234` - C-style hex
- `0Fh`, `1234h` - Intel assembly style
- `#FFF`, `#FFFFFF`, `#FFFFFFFF` - CSS colors with alpha

### Wiki Link Format

MediaWiki-style subpage links:
```
[[Game Name:Subpage|Display Text]]
```

Example:
```
[[Dragon Warrior II:ROM_Map|ROM Map]]
[[The 7th Saga:SRAM_Map|SRAM Map]]
```
