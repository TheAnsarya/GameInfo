# Dragon Warrior III (NES) Documentation

## Overview

- **Platform:** Nintendo Entertainment System (NES)
- **Developer:** Chunsoft
- **Publisher:** Enix
- **Year:** 1988 (Japan), 1991 (US)
- **Mapper:** MMC5 (mapper 5)
- **PRG ROM:** 512KB (32 Ã— 16KB banks)
- **CHR ROM:** 256KB
- **Battery Save:** Yes

## Key Features

- Class/vocation system with 9 classes
- Personality system affecting stat growth
- Day/night cycle
- Two world maps (Light and Dark worlds)
- Party of up to 4 characters (Hero + 3 custom)
- Class change system at Dharma Temple
