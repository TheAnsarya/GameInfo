## Supported Formats

All formats support both read and write operations:

| Format | Emulator | System |
|--------|----------|--------|
| SMV | Snes9x | SNES |
| BK2/BKM | BizHawk | Multi |
| LSMV | lsnes | SNES |
| MMO | Mesen2 | Multi |
| FM2/FCM/FMV | FCEUX/Famtasia | NES |
| VBM | VisualBoyAdvance | GB/GBA |
| GMV | Gens | Genesis |
| MMV | Dega | SMS/GG |
| PXM/PJM | PCSX/PSXjin | PSX |
| YMV | Yabause | Saturn |
| MC2 | PCEjin | PCE |
| DSM | DeSmuME | NDS |
| LMP | Doom | DOS |
